Purpose

if you want to add css style to your own html , how ?

  1. find some css
  2. refer to current folder more info
  3. type <link> tag in <head> tag
    • the parameter rel="stylesheet" (rel = relation)
    • the parameter type="text/css"
    • the parameter href="where is the css located"
  4. have some fun with the page you've created !
  5. if you still have any questions just go to codecademy in the project that you've created or just find some in google (maybe chatgpt?)
